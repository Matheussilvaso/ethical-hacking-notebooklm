# 🛡️ Ethical Hacking com NotebookLM

Aprendizado ativo com Inteligência Artificial aplicado à Segurança da Informação e Ethical Hacking.

---

## 🎯 1. Contexto e Objetivos

Este projeto tem como objetivo construir um **Caderno Temático utilizando o NotebookLM**, focado em **Ethical Hacking (Hacking Ético)**.

A proposta é utilizar IA como ferramenta de apoio para aprendizado estruturado, análise crítica e organização do conhecimento na área de segurança cibernética.

### 📌 Objetivos de estudo:

- Compreender os fundamentos do Ethical Hacking;
- Identificar as principais técnicas utilizadas por hackers éticos;
- Explorar ferramentas e metodologias de testes de segurança;
- Analisar riscos, vulnerabilidades e boas práticas;
- Consolidar o aprendizado em um guia reutilizável.

---

### 🧠 4.1 Resumos Estruturados

- Ethical Hacking consiste em identificar vulnerabilidades em sistemas de forma legal e controlada;
- Atua na prevenção de ataques cibernéticos;
- Principais áreas:
  - Pentest (teste de invasão)
  - Engenharia social
  - Análise de vulnerabilidades
- Ferramentas populares:
  - Nmap
  - Metasploit
  - Wireshark
  - Burp Suite
- Metodologia comum:
  - Reconhecimento
  - Varredura
  - Exploração
  - Pós-exploração
- Importância:
  - Proteção de dados
  - Segurança corporativa
  - Prevenção de fraudes

---

### 📖 4.2 Glossário

- **Pentest** → Simulação de ataque para encontrar falhas  
- **Exploit** → Código usado para explorar vulnerabilidade  
- **Firewall** → Sistema de proteção de rede  
- **Phishing** → Ataque de engenharia social  
- **Malware** → Software malicioso  
- **Zero-day** → Vulnerabilidade desconhecida
